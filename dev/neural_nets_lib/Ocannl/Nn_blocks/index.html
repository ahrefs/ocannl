Module Ocannl.Nn_blocks

This file contains basic building blocks for neural networks, with limited functionality. Feel free to copy-paste and modify as needed.

We follow "the principle of least commitment": where possible, we use row variables to remain agnostic to the number of axes. This flexibility often remains unused, but it makes explicit the architectural structure.
